# Пиратский Морской Бой (Pirate Sea Battle)

Standalone fullscreen web game — пиратский морской бой с полноценным меню, двумя режимами игры и ретро-графикой.

## 🎮 Играть

[https://7dzmycyb7vhl2.kimi.page](https://7dzmycyb7vhl2.kimi.page)

## 📋 Главное меню (4 пункта)

### 1. ⚔ Бой с ботом
- **Выбор сложности**: Лёгкий / Сложный (←→)
- **Выбор расстановки**: Авто / Вручную (←→)
- **Лёгкий бот**: стреляет случайно
- **Сложный бот**: шахматный поиск + добивание раненых кораблей
- Навигация: ↑↓ — выбор строки, ←→ — изменение, Enter — старт

### 2. 👥 1 на 1
- Два игрока на одном ПК
- Поочерёдная расстановка кораблей
- Поочерёдные ходы с таймером 3 секунды
- Экран передачи хода между игроками

### 3. ⚙ Настройки
- Информация об игре
- Управление
- Правила

### 4. 🚪 Выход

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `↑` `↓` | Навигация / Перемещение курсора |
| `←` `→` | Изменение опции / Перемещение |
| `Enter` | Подтвердить / Выстрел / Установить |
| `Space` | Поворот корабля |
| `R` | Авто-расстановка |
| `Escape` | Назад |
